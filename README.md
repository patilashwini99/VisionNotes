# Vision Notes App 📒

This is an Android Notes Application developed using Java and Android Studio.

## 📱 Features
- Add Notes
- Edit Notes
- Delete Notes
- Simple and user-friendly UI

## 🛠️ Tech Stack
- Java
- Android Studio
- XML
- SQLite / Room Database (if used)

## 🚀 How to Run
1. Clone this repository
2. Open in Android Studio
3. Sync Gradle
4. Run on Emulator or Mobile Device
